# 发布说明

## 是否需要安装 Python / PyQt

不需要。

当前发布包已经包含：

- Python 运行时
- PyQt6 模块
- Qt6 DLL 与平台插件
- VC 运行时 DLL

因此在 64 位 Windows 10 / 11 新装系统上，一般不需要再安装 Python、PyQt6 或 Visual C++ 运行库。

## 推荐给别人使用哪个版本

优先推荐：

- `release\Portable\MonitorInsightPortable`

让对方完整保留整个文件夹结构，然后双击其中的：

- `MonitorInsight.exe`

这是最稳的便携版发布方式。

## 单文件版

如果你想发一个单独 EXE，也可以使用：

- `release\OneFile\MonitorInsight.exe`

但单文件版首次启动会先解包，且更容易受杀毒软件、权限、临时目录策略影响。
因此对于“刚装好的系统”，仍然建议优先发便携版文件夹或对应 zip。

## 可直接发送的压缩包

- `release\MonitorInsight_Portable.zip`
- `release\MonitorInsight_OneFile.zip`

## 目标环境

- Windows 10 / 11 64 位
- 建议本机已安装常规显卡驱动
- 信号切换功能要求显示器开启 DDC/CI
